Adds the possibility to set an upublish date to pages, posts and custom post types.

== Description ==

The plugin adds a datetime box to your edit pages to set a unpublish date to posts, pages, and custom post types.

After you set a unpublish date the post will we updated at the given time and become a draft.
The plugins works with posts, pages and custom post types

== Installation ==

1. Upload the 'gtuk-unpublish-posts' plugin to your '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= The post has not been updated =

Make sure you set the correct timezone.

= Can i add the unpublish date to my custom post types =

The plugin works with your custom post types out of the box.
